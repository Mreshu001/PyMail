Overview

PyMail is a single-file, interactive CLI application for reading and composing email entirely in your terminal. It leverages:

  • IMAPClient for folder navigation, threaded message lists, and search
  • smtplib for sending new messages with attachments
  • python-gnupg for inline PGP/GPG signing and encryption
  • prompt_toolkit for menu navigation and customizable keybindings
  • Rich for beautiful tables, syntax-highlighted markdown, and notifications

Features

  • IMAP folder navigation and threaded view
  • Search messages by subject or sender
  • Compose messages in your editor with attachment support
  • Inline PGP/GPG signing and encryption of outgoing mail
  • Download and upload attachments
  • Customizable shortcuts via config
  • Color themes for prompt_toolkit
  • Persistent config in ~/.config/pyemail/config.yaml

Prerequisites

  • Python 3.8 or newer
  • IMAP and SMTP email account
  • GPG with a keypair
  • Terminal with 256-color support (optional)

Installation

git clone https://github.com/bocaletto-luca/PyMail.git
cd PyMail
pip install -r requirements.txt
chmod +x pymail.py

Configuration

On first run, PyMail creates a default config file at ~/.config/pyemail/config.yaml. Edit it with your server and account settings:

imap_host: imap.example.com
imap_port: 993
smtp_host: smtp.example.com
smtp_port: 587

email: user@example.com
gpg_recipient: YOUR_GPG_KEYID

shortcuts:
  quit: q
  back: b

theme: monokai

Usage

  1. Run ./pymail.py and enter your password.
  2. Select Folders to browse mail folders and read messages.
  3. Select Search to find mails by subject/from.
  4. Select Compose to draft and send new mail (with PGP support).
  5. Select Quit to exit the client.

Shortcuts & Themes

Customize keys and UI colors in config.yaml under shortcuts and theme.
